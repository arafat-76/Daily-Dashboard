# Daily Dashboard

A Python command-line app that shows your daily weather and a motivational quote.

## Features
- Live weather for any city
- Random motivational quote
- Saves data to dashboard.json

## How to use
1. Install libraries: pip install requests beautifulsoup4
2. Run: python dashboard.py
3. Enter your city name

## Built with
- Python
- requests library
- BeautifulSoup4
- wttr.in API
- quotes.toscrape.com
